What 's ChatManager?

ChatManager Features

Chat Message Monitors

The ChatManager monitors are a super fast way to trigger actions based on the contents of a chat message. For example, monitoring for questions, requests for live person, prompt injections, message sentiment etc.

Chat Session Recording
